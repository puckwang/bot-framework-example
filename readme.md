# bot-framework-example
用於練習寫 bot 並做個紀錄。

### Command
* Cards: Demo 卡片
* Hello: Demo State
* Post: Demo 取外部 API 資料
* Operation / Operation-v2: 簡易計算，v2 為改使用 form flow 的寫法
